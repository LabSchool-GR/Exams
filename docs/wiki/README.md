# Wiki Starter Pack

These pages are prepared as a clean starter structure for the GitHub Wiki of the `Exams` repository.

You can use them in two ways:

1. Keep them here as versioned documentation inside the repo.
2. Copy each file into the matching page of the GitHub Wiki.

Greek companion pages are also available under `docs/wiki/el/`.

Suggested page order:

- `Home.md`
- `Installation-and-Setup.md`
- `Features-and-User-Flows.md`
- `Operations-and-Releases.md`
- `Testing-and-Quiz-Templates.md`
- `Security-Privacy-and-Compliance.md`

Suggested Wiki page names:

- `Home`
- `Installation and Setup`
- `Features and User Flows`
- `Operations and Releases`
- `Testing and Quiz Templates`
- `Security Privacy and Compliance`

The content intentionally points back to the repo docs so the wiki stays short and readable, while the detailed operational material remains versioned with the codebase.
