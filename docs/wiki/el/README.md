# Greek Wiki Starter Pack

These pages are the Greek companion set for the repository wiki starter pack.

Suggested usage:

1. Review and edit them locally first.
2. Copy them into the GitHub Wiki as separate Greek pages.
3. Optionally keep both Greek and English pages in the wiki.

Suggested Greek Wiki page names:

- `Αρχική`
- `Εγκατάσταση και Ρύθμιση`
- `Λειτουργίες και Ροές Χρηστών`
- `Λειτουργία και Εκδόσεις`
- `Δοκιμές και Πρότυπα Quiz`
- `Ασφάλεια Απόρρητο και Συμμόρφωση`

These files intentionally use ASCII filenames locally, while their content is fully Greek.
