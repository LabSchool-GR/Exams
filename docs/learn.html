Open-source application

LabSchool Exams

Exams is a Laravel-based knowledge assessment application designed for schools, training providers, and public or private educational programs.

Home Page Try the application Sample Quiz Support Project

01. Teacher Registration

Create a teacher account, sign in securely, verify access, recover passwords, and enter the application workspace.

02. Quiz Creation

Build quizzes, set language, category, timing, scoring, access rules, visibility, images, and publication options.

03. Quiz Questions

Create questions, define possible answers, choose correct answers, import content, and organize the quiz question bank.

04. Examinee Management

Manage participants, student lists, personalized links, PIN access, guest participation, and anonymous access modes.

05. Quiz Display Templates

Select visual templates for the participant experience, including classic, image-based, and event-style quiz layouts.

06. Quiz Execution and Flow

Understand how participants start, answer, navigate, submit, resume, and complete a quiz under the configured rules.

07. Quiz Results

Review attempts, scores, pass status, question statistics, exports, reports, PDFs, certificates, and verification links.

08. Administrator Options

Control users, quotas, updates, system settings, operational checks, and administrative access across the installation.

09. Teacher Options

Use the teacher workspace for quizzes, questions, participants, reports, profile settings, feedback, and everyday classroom workflows.
